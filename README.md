# Embedded Systems Lab, Fall 2017
## For Embedded Systems Labs 2017 Fall homeworks, I2C and BLE programs.
## Usage
### I2C
#### rpi與arduino連線
```
在`I2C`資料夾中

在auduino端執行i2c-Arduino.c

在rpi3端執行i2c-RPi3.c或i2c-RPi3.py
  
等待雙方連線
```
#### 讀取ADXL345數值
```
在ADXL345資料夾中

在rpi3上執行adxl345-RPi3.c或adxl345-RPi3.py

在終端機顯示讀取的三軸加速度值
```
